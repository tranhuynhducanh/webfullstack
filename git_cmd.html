1. Git status
          - trạng thái hiện tại của respo mà mình đang đứng
         2. Git branch 
          - Tạo 1 nhánh mới từ nhánh hiện thành
         3. Git checkout 
         - Chuyển trạng thái qua 1 nhánh đào tạo
         4. git commit -m ""
         - tạo 1 cam kết khi chúng xác 1 file hoặc nhiều file thay đổi
         5. git push origin 
         - Đẩy những thay đổi mới nhất mà chúng ta đã cam kết lên origin
         6. git add 
         - đẩy những thay dổi mới nhất mà chúng ta cam kết lên origin
